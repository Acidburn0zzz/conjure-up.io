Start using your big software instead of learning how to deploy it

Conjure-up lets you summon up a big-software stack as a “spell” — a model of the stack, combined with extra know-how to get you from an installed stack to a fully usable one.

Get started

  • $ sudo snap install conjure-up classic
  • $ conjure-up

Conjure-up spells

Kubernetes

$ conjure-up kubernetes

Get the Canonical Distribution of Kubernetes deployed and running all on a single machine.

Kubernetes with conjure-up

openstack

$ conjure-up Openstack

Easily deploy real-world OpenStack on a single machine using LXD containers.

Openstack with conjure-up
